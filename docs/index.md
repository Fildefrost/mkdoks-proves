# 🧠 Blog de Ciberseguridad

<p class="mb-5">
Bienvenid@ a mi blog! Este sitio ha sido creado para publicar mi trayectoria y aprendizaje en
el campo de la ciberseguridad y el hacking ético. Aquí iré subiendo tanto documentación
técnica como resolución de entornos vulnerables (CTF) con el único objetivo de documentar mi
progreso.

encantará saberlo.
</p>
